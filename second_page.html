HTML CSS JS

HTML (от английского HyperText Markup Language) — это язык гипертекстовой разметки текста.
  • Отвечает за структуру содержимого веб-страницы,
  • Позволяет определить элементы и их иерархические отношения.
  • Используется для создания текстового и мультимедийного контента на веб-страницах.
  • Примеры элементов HTML: < div>, < p>, < img>, < a>, < ub>, < li>.
CSS — это код, который позволяет визуально оформить страницу: раскрасить подзаголовки, поменять фон или отформатировать изображение. CSS — важная часть фронтенд-разработки и один из ключевых навыков для веб-разработчика. В статье расскажем, как устроен код, где используется и как с помощью несложного и понятного CSS-программирования сделать первые шаги в IT-профессии.
  • Отвечает за внешний вид и оформление веб-страницы.
  • Позволяет задавать стили для элементов, такие как цвет текста, размеры и отступы.
  • Обеспечивает разделение структуры и внешнего вида, что упрощает обслуживание кода.
  • Примеры CSS свойств: color, font-size, margin, padding, background.
JavaScript (JS) — это интерпретируемый и мультипарадигменный язык программирования.
  • Отвечает за динамическое поведение веб-страницы и интерактивность.
  • Позволяет добавлять функциональность, такую как анимации, обработка событий, валидация форм и обмен данными с сервером.
  • Может изменять содержимое и структуру страницы, обновлять данные без перезагрузки страницы и многое другое.
  • Примеры использования JavaScript: добавление элементов страницу, валидация формы, обработка кликов и т. д.
